## What is Worlds?

Worlds is an open-source **neuro-symbolic infrastructure layer** that gives AI agents persistent, verifiable memory. Instead of relying on probabilistic semantic search alone, Worlds combines symbolic logic (SPARQL knowledge graphs) with neural vector search to give your agents deterministic, structured memory.

## Why neuro-symbolic?

Traditional RAG patterns rely on similarity search, which causes agents to hallucinate or lose structural context. Worlds bridges this "Reasoning Gap" by fusing hybrid search with symbolic logic.

| Feature | Semantic RAG | Worlds |
| :------ | :----------- | :----- |
| **Recall type** | Probabilistic intuition | Deterministic precision |
| **Logic** | Implicit weight-based logic | Explicit RDF-based logic |
| **Querying** | Similarity search | SPARQL 1.1 + Hybrid Search |
| **State** | Ephemeral / Static | Malleable / Verifiable |

## Get started

<Steps>
  <Step title="Install the SDK">
    ```bash
    deno add jsr:@wazoo/worlds-sdk
    ```
  </Step>
  <Step title="Initialize the client">
    ```typescript
    import { WorldsSdk } from "@wazoo/worlds-sdk";

    const sdk = new WorldsSdk({
      baseUrl: "http://localhost:8000",
      apiKey: "your-api-key",
    });
    ```
  </Step>
  <Step title="Add knowledge to your world">
    ```typescript
    await sdk.worlds.sparql(
      "my-knowledge-base",
      `INSERT DATA {
        <http://example.com/alice> <http://schema.org/knows> <http://example.com/bob> .
        <http://example.com/bob> <http://schema.org/givenName> "Bob" .
      }`
    );
    ```
  </Step>
  <Step title="Query and reason">
    ```typescript
    const result = await sdk.worlds.sparql(
      "my-knowledge-base",
      `SELECT ?name WHERE {
        <http://example.com/alice> <http://schema.org/knows> ?person .
        ?person <http://schema.org/givenName> ?name .
      }`
    );
    ```
  </Step>
</Steps>
